# Política de Seguridad

# Fuente Oficial y Canónica
ADVERTENCIA CRÍTICA: La única fuente confiable y oficial para descargar *Anestesia* es el repositorio original en GitHub:
> **https://github.com/celestayelen-tech/anestesia**

Cualquier binario descargado de otros sitios, foros, repositorios clonados o enlaces de terceros *NO está soportado* y podría haber sido manipulado (contener malware).
*La autora no se hace responsable de ejecuciones fuera de este entorno oficial.*

# Versiones compatibles
Utilice la última versión (v0.2.0+). Las versiones anteriores no son compatibles debido a cambios en la API de Windows y no reciben parches de seguridad.

# Integridad (Verificación Obligatoria)
Para garantizar que tiene el código oficial de mi autoría:
1. Vaya a la sección *Releases* en el repositorio oficial mencionado arriba.
2. Descargue el archivo `.exe`.
3. Copie el Hash SHA-256 publicado en las notas de la versión.
4. Verifique su archivo localmente. Si el hash no coincide exactamente, *BORRE EL ARCHIVO INMEDIATAMENTE*

# Informar una vulnerabilidad
Si encuentra un error que comprometa la estabilidad o seguridad del sistema, abra un *Issue* en este repositorio. Esta herramienta está diseñada para ser *fail-open*: si falla, debe dejar el sistema en un estado estándar, no roto.
